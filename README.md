# weka

Execute Weka GUI
```
$ mvn package exec:java -Dexec.mainClass=weka.gui.GUIChooser
```

Execute sample
```
$ mvn clean package exec:java -Dexec.mainClass=com.neetomo.weka.Titanic
```
